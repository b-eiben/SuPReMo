Introduction

SuPReMo implements the unified motion modelling and image registation framework as proposed by Jamie McClelland et al. and was developed by Bjoern Eiben and the Radiotherapy Image Computing Group at the Centre for Medical Image Computing, University College London, UK.

The source code is available on github and the documentation is hosted on the corresponding github-pages.
